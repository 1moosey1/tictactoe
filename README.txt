Tic Tac Toe

Built based on the Tic Tac Toe challenge from freeCodeCamp
https://www.freecodecamp.com/challenges/build-a-tic-tac-toe-game


Details:

You can choose between X's and O's
Player will always go first
Opponent is the computer (alpha-beta prune)
- The algorithm used for the opponent makes it unbeatable
- The goal was not to make a beatable computer but to practice with unfamiliar algorithms

Technologies:
HTML, CSS, JavaScript, jQuery
